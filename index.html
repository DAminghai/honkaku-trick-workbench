导出 / 备份

localStorage 偶尔会被浏览器清掉，定期导出 JSON 才安全（含书目、标签、拆解、图谱、关系网、灵感、自定义分区与技法）。

导入 / 还原

从导出的 JSON 恢复全部数据（覆盖当前）。换新版本文件时，先在旧文件导出、再到这里导入。

自定义分区管理

在添加 / 编辑书目时，分区下拉里选「+ 新建分区…」即可创建。已建的自定义分区在此重命名或删除（删除后归入「未分类」）。

关于

书单数据在 books.js，与应用逻辑分离，便于增删与协作。图谱用到联网加载的 vis-network（首次需联网，之后缓存）。更稳的存储：跑 python -m http.server 后访问 http://localhost:8000/。
