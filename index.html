使用指南

诡计拆解工作台 · 创作者深度指南

工作台的核心哲学是：将阅读过的被动输入，转化为结构化的、可随时调用的写作弹药库。整个系统围绕着“读 → 拆 → 连 → 找灵感”的闭环运转。

一、 书单管理：建立你的素材坐标系

  • 分区（Section）的用法：相当于书籍的主分类。你可在添加时随时“+ 新建分区”。
    实战技巧：例如为你正在构思的“生辰密室”建一个专属的资料分区，将相关的参考书籍全部收拢其中。
  • 题材/分类标签（Tags）：跨越分区的细粒度标记。
    实战技巧：标签应当具体化，如“建筑-重力机关”、“民俗-道教”、“童谣杀人”，以便后期按图索骥提取素材。

二、 拆解卷宗：外科手术式的诡计剖析

  • 点击书目的“拆解 +”进入。这里不是写读后感，而是进行反向工程。
  • 线索布置与误导：重点剖析前人是如何把真线索藏在视线盲区，又如何利用红鲱鱼吸引读者注意力的。
  • 可借鉴点：明确写下该手法中哪一部分可以直接“偷师”或进行变种改造（例如：“这个利用重力的物理机关，可以改造成利用道教五行相克中的水流来触发”）。

三、 图谱与关系网：侦探白板与诡计演化史

  • 单本图谱（Board）：内置的推理白板。当遇到如《首无》般多谜题并发的巨著时，可用来复盘三维建筑平面图或是真假解答的推导树。
  • 跨书关系网（Cross Network）：避免撞梗与寻找升级思路的利器。圆点代表书籍，连线代表手法上的血脉关系。点击“＋ 连接模式”即可建立羁绊。
    实战技巧：把两本同样使用了“时间诡计”的书连起来，在备注中写上它们各自的掩盖手法，借此反向思考你能否开创出第三种掩盖方式。

四、 终极武器：灵感矩阵与灵感箱

  • 灵感矩阵（Insight Matrix）：行=技法标签，列=题材标签。有数字的格子代表前人已有的组合，带有“＋”号的空格则是无人涉足的处女地。只需轻轻一点，即可直接在这个空白坐标上生成一张原创灵感卡！
  • 灵感箱（Idea Box）：记录你灵光一闪的诡计原型（例如：利用二十四节气的交替温差制造延时密室），并随时绑定那些为你带来启发的源头书目。

五、 数据安全与防丢指南（必读）

  • 本工作台是纯本地、零后端的静态工具。这意味着它极速且绝对隐私安全，但代价是所有数据仅存储在你当前浏览器的缓存中。
  • 如果你清除了浏览器缓存，或者使用了无痕/隐身模式关闭窗口，数据将会永久丢失。请务必养成每次大量拆解后，去「数据」页点击“导出 JSON 备份”的良好习惯。
  • 当你在另一台设备上使用，或获得了新版本的 HTML 文件时，只需在新环境的「数据」页导入你的 JSON 备份，一切心血即可瞬间无损恢复。你还可以利用“导出为 Markdown”功能，将素材库整装搬入 Obsidian 或 Notion。
